# brightspace-course-demo

HTML snippets and interactive components for MLRI Brightspace courses — wrappers, callouts, tabs, accordions, flip cards, and related topic pages.

Part of the LACE + Brightspace stack. See `LACE-Brightspace-Plan.md` for architecture context.

## Contents

- **Course Wrapper.html** — course shell and navigation
- **Interactive elements** — Accordion, Tabs, Callouts, Flip Cards, Fill-in-the-Blanks, Sequencing, Sorting Gallery, Stylized Quotes
- **LACE-Login-Page-Snippets.html** — branded Brightspace login page (Message + Additional Content)
- **Planning docs** — `LACE-Brightspace-Plan.md`, `MVP-MVM-Requirements.md`

Paste HTML into Brightspace topic pages or Login Page Management (Source view) as noted in each file.
